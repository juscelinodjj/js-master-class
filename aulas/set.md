# Set

- Um Set é um objeto que armazena **elementos únicos**, que podem ser de qualquer tipo de dado.
- **size**
  - Retorna a quantidade de elementos.
- **add**
  - Adiciona um elemento
- **forEach**
  - Itera sobre o set.
- **has**
  - Retorna true se o elemento existir.
- **delete**
  - Remove um elemento.
- **clear**
  - Remove todos os elementos.
